快速指南
  1. 在「域名管理」中添加你的邮件域名（如 example.com）
  2. 在「邮箱管理」中创建用户邮箱（user@example.com）
  3. 在「端口管理」中查看/自定义 SMTP/POP3/IMAP 端口
  4. 在域名 DNS 中添加对应的 MX/A/TXT 记录（域名详情页可查看）
  5. 执行 bash scripts/start-services.sh 启动邮件协议服务
  6. 使用任意邮件客户端 (如 Outlook/Foxmail) 即可收发邮件
  7. 通过「API 密钥」生成 Token 进行第三方对接
